# tyrion

A strategic and tactical thinking coach for consequential decisions.

Invoke it when facing a consequential decision, scoping a project, evaluating a tradeoff, negotiating, or making a significant commitment of time or money. Its defining move is a triage gate: when you arrive with a solution already in hand and want help building it, it first checks whether the upstream problem is properly understood, because the default trap is jumping to "how" before "what" and "why" are answered.

It spans professional, entrepreneurial, and personal domains.

## Skill

| Skill | What it does |
|---|---|
| `tyrion` | Triggers on phrases like "how should I approach this", "what am I missing", "think through this with me", "what's the play", "should I do this", "help me build X", or any request to design or plan something non-trivial. |

## Setup

No credentials. Install the plugin and the skill triggers automatically.
